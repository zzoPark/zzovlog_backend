# zzovlog
The backend of zzovlog
## 개발환경
- Python, Django, Django Rest Framework 기반
- CORS error 해결을 위해 [django-cors-headers](https://github.com/ottoyiu/django-cors-headers)도 이용함
- Apache + mod_wsgi
